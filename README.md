# PyPassword Generator

A simple Python-based password generator that creates secure, random passwords using user-defined combinations of letters, numbers, and symbols.

## Features

- Customisable password length  
- User-defined number of:
  - Letters  
  - Symbols  
  - Numbers  
- Fully randomised output using list shuffling  
- Beginner-friendly implementation using core Python concepts  

## What This Project Demonstrates

This project was built as part of learning Python fundamentals and showcases:

- For loops and iteration  
- Conditional thinking and logic flow  
- Working with lists  
- Random selection using `random.choice()`  
- Shuffling data with `random.shuffle()`  
- Converting data types (list → string)  

## How It Works

1. User inputs how many letters, symbols, and numbers they want  
2. The program randomly selects characters from predefined lists  
3. All characters are stored in a list  
4. The list is shuffled to randomise order  
5. The final password is generated and displayed
